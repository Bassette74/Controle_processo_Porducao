
README - Controle de Processo de Produção

Dashboard Multi-Projeto em Streamlit para gestão visual de cronogramas, etapas e status.
Inclui funcionalidades como edição de fases, importação/exportação de Excel, cálculo automático de status, verificação de sobreposição e visualizações profissionais com gráficos Gantt e barras.

Funcionalidades Principais:
- Gerenciamento de múltiplos projetos
- Adicionar, editar, excluir e finalizar fases
- Importar e exportar planilhas Excel
- Cálculo inteligente de status (Dentro do Prazo, Quase Atraso, Atraso, etc.)
- Detecção de sobreposição de datas
- Gráficos Gantt e gráficos de barras com Plotly
- Interface moderna com Streamlit e possibilidade de uso de CSS

Tecnologias:
- Python 3
- Streamlit
- Pandas
- Plotly
- OpenPyXL

Este sistema é ideal para equipes de produção, engenharia, PCP, logística e gestão de projetos que necessitam de acompanhamento visual claro e eficiente de processos.
